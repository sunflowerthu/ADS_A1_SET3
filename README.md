# Задача A1 из SET 3 по предмету "Алгоритмы и структуры данных"
В данном репозитории представлены исходные данные эксперимента, код алгоритма для codeforces, код алгоритма для эксперимента, а также код на Python, необходимый для построения графиков. Также предсатвлены сами графики.

### Навигация по репозиторию:
Все файлы находятся в папке files.
- codeforces.cs - код алгоритма, проверенный на codeforces.
- experiment.cs - код алгоритма для эксперимента, который записывает результаты в csv-файл.
- experiment_data.csv - результаты эксперимента в формате csv.
- area_comparison.png - график, показывающий разницу между широкой и узкой областями.
- error_comparison.png - график, показывающий зависимость ошибки от количества точек и типа области.
- A1_SET3.pdf - отчёт по задаче с выводом.
